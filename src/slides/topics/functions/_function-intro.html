@function turns CSS into a toolkit

The proposal lets us author callable helpers with parameters, default values, and returns. We finally reuse the same math without copy and paste. 
          
          @function flux-glow($energy, $base: 8px) {
            @return clamp(4px, $energy * $base, 32px);
          }

          .flux-panel {
            box-shadow: 0 0 flux-glow(attr(data-energy number, 1))
              rgba(0, 245, 255, 0.4);
          }
Helper output

One helper defines the glow thickness.
@function is in CSS Values 5 and will feel natural once typed attr feeds it.
