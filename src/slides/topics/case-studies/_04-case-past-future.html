Case: Past vs future stories on the same markup

Marketing wanted two visuals. We only swapped data-era. 
        
        <hero-card data-era="past" data-speed="65"></hero-card>
        <hero-card data-era="future" data-speed="88"></hero-card>
      
        
        hero-card[data-era='past'] {
          background: linear-gradient(135deg, #451b03, #120502);
        }

        hero-card[data-era='future'] {
          background: linear-gradient(135deg, #041e3f, #00121f);
        }
Hero card

Copy, glow, even imagery switched via CSS attr selectors.
Result: no duplicate components, only theme-level CSS.
