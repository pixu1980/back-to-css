Case: Flux dashboard for mission control

Attributes orchestrate dozens of widgets with zero JS toggles while keeping accessibility intact. 
        
        <section class="flux-dashboard"
          data-era="future"
          data-phase="jump"
          data-stop="2015"
        >
          <widget-card data-widget="flux" data-energy="1.21"></widget-card>
          <widget-card data-widget="timeline" data-stop="2015"></widget-card>
        </section>
      
        
        .flux-dashboard {
          --accent: if(attr(data-era token) = future, var(--btf-teal), var(--btf-orange));
          color: var(--accent);
        }

        .widget-card[data-widget='flux'] {
          box-shadow: 0 0 flux-glow(attr(data-energy number));
        }
