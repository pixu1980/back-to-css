Timeline controls feed CSS math

One slider updates data-stop which cascades into glow, labels, and scroll timelines. 
        const slider = document.querySelector('[data-timeline]');
slider.addEventListener('input', (event) => {
  const year = event.target.value;
  event.target.parentElement.setAttribute('data-stop', year);
});
        [data-stop] .timeline-marker {
  transform: translateX(timeline-offset(attr(data-stop number)));
}

[data-stop] .status::after {
  content: 'ETA ' attr(data-stop);
}
