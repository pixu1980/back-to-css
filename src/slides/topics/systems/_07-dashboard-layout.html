Dashboards composed from attribute props

Cards do not know anything about their parents. They simply react to the attributes they receive. 
        <section class="dashboard" data-era="future" data-mode="monitor">
  <card data-widget="timeline" data-speed="88"></card>
  <card data-widget="power" data-energy="1.21"></card>
</section>
        .dashboard {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
}

[data-widget='power'] {
  --flux-energy: attr(data-energy number, 1);
}
