Forms set attributes directly

Inputs update data-speed and data-era through form participation. CSS listens. 
        const form = document.querySelector('#era-form');
form.addEventListener('input', (event) => {
  const panel = document.querySelector('#demo-systems-form');
  panel.setAttribute(event.target.name, event.target.value);
});
        #demo-systems-form {
  --flux-era-hue: if(attr(data-era token) = future, 196, 28);
  --flux-speed: attr(data-speed number, 60);
}

#demo-systems-form .status::after {
  content: 'Era ' attr(data-era) ', ' attr(data-speed) ' mph';
}

Form inputs are attribute editors. CSS logic reads the current value.
Use form participation to keep everything accessible (labels, keyboard support).
