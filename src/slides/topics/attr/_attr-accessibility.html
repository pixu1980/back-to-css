Mirror attributes to accessibility

When attributes are the truth, assistive tech can read the same stream. Mirror the value into aria-live, announce mode changes, and keep the UI inclusive even when the CSS gets fancy. 
          
          <section
            class="flux-panel"
            data-era="future"
            aria-live="polite"
            aria-label="Flux mode future">
          </section>
        
          
          .flux-panel::before {
            content: attr(data-era);
          }

          @media (prefers-reduced-motion: reduce) {
            .flux-panel .flux-radial {
              animation: none;
            }
          }
