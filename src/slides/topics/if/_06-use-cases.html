Powerful Use Cases

From theming to animations, if() handles it all.

Dynamic theming

    
    :root {
      --theme: attr(data-theme token, light);

      --bg: if(var(--theme) = dark, #0a0a0f, #ffffff);
      --fg: if(var(--theme) = dark, #f0f0f0, #1a1a1a);
      --accent: if(var(--theme) = dark, #00f5ff, #0066cc);
      --surface: if(var(--theme) = dark, #1a1a2e, #f5f5f5);
    }

    body {
      background: var(--bg);
      color: var(--fg);
    }

Responsive breakpoint values

    
    .grid {
      --cols: attr(data-cols number, 1);

      /* Clamp columns based on container */
      grid-template-columns: repeat(
        if(var(--cols) > 4, 4, var(--cols)),
        1fr
      );

      /* Adjust gap */
      gap: if(var(--cols) >= 3, 2rem, 1rem);
    }

Animation control

    
    .notification {
      --priority: attr(data-priority token, normal);

      animation-name: if(
        var(--priority) = urgent,
        pulse-urgent,
        fade-in
      );

      animation-duration: if(
        var(--priority) = urgent,
        0.3s,
        0.6s
      );

      animation-iteration-count: if(
        var(--priority) = urgent,
        3,
        1
      );
    }
