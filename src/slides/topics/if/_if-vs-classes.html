if() replaces helper classes

Instead of flipping .is-active everywhere, evaluate the condition once. The CSS cascade handles the rest, even inside gradients or transforms where classes cannot reach. 
          // before
          button.addEventListener('click', () => {
            panel.classList.toggle('is-critical', speed >= 88);
          });
        
          
          .flux-panel {
            --glow-alpha: if(attr(data-speed number, 0) >= 88, 0.7, 0.25);
            box-shadow: 0 0 35px rgba(0, 245, 255, var(--glow-alpha));
          }
Classless diff

Glow intensifies when the condition flips.
No helper classes, no manual cascade ordering. The condition lives next to the property it controls.
