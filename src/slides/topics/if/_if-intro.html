Native if() brings logic to CSS

The ternary syntax moves into CSS so we can branch on numbers, tokens, or booleans directly in the cascade. No more helper classes for simple checks. 
          
          .flux-panel {
            --status-hue: if(attr(data-speed number, 0) >= 88, 140, 28);
            box-shadow: 0 0 25px hsl(var(--status-hue) 80% 60%);
          }

          .flux-panel .status::after {
            content: if(
              attr(data-speed number, 0) >= 88,
              'Jump ready',
              'Charging'
            );
          }
