Chain if() for multi step logic

Combine conditions to create tiers: charging, warmup, and jump ready. The CSS stays declarative, yet the logic rivals a small reducer. 
          
          @property --flux-phase {
            syntax: '<integer>';
            inherits: false;
            initial-value: 0;
          }

          .flux-panel {
            --flux-phase: if(attr(data-speed number, 0) >= 88, 2,
              if(attr(data-speed number, 0) >= 60, 1, 0)
            );
          }

          .flux-panel .status {
            color: if(var(--flux-phase) = 2, #00f5ff,
              if(var(--flux-phase) = 1, #ff9500, #9ba0c4)
            );
          }
