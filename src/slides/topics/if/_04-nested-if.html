Nested if() Chains

Build complex logic by nesting conditions.

Multi level branching

    
    .status-indicator {
      --level: attr(data-level number, 0);

      /* 4-tier status */
      --status-color: if(var(--level) >= 100, #00ff88,
                      if(var(--level) >= 66, #88ff00,
                      if(var(--level) >= 33, #ffaa00,
                         #ff4444)));

      --status-label: if(var(--level) >= 100, 'Full',
                      if(var(--level) >= 66, 'Good',
                      if(var(--level) >= 33, 'Low',
                         'Critical')));

      background: var(--status-color);
    }

    .status-indicator::after {
      content: var(--status-label);
    }

Combining multiple checks

    
    .dashboard {
      --speed: attr(data-speed number, 0);
      --energy: attr(data-energy number, 0);

      /* Both conditions must be true */
      --is-ready: if(
        var(--speed) >= 88,
        if(var(--energy) >= 1.21, true, false),
        false
      );

      /* Use the combined result */
      animation: if(
        var(--is-ready) = true,
        time-warp 0.5s ease-out,
        idle 2s infinite
      );
    }
